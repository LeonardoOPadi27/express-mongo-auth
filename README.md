# Mi Proyecto

Este es un proyecto de ejemplo para probar el despliegue automático con GitHub Actions y Render.

## 🚀 Despliegue automático

Cada vez que se hace push a la rama `main`, GitHub Actions ejecuta el workflow `Deploy to Render` y despliega automáticamente la aplicación.

## 📦 Tecnologías usadas

- GitHub Actions (CI/CD)
- Render (Hosting)
- Git

## 🧪 Probar el despliegue

1. Hacer cambios en el código
2. Hacer commit y push a `main`:
   ```bash
   git add .
   git commit -m "README actualizado"
   git push origin main
